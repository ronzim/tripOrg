# triporg
A simple tool for organizing trip memories. Built with vue-cli-plugin-electron-builder.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn electron:serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
